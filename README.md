personal-expense-tracker
A Java-based application to help users manage and track personal expenses. Demonstrates object-oriented programming, GUI development using Swing, and basic data management.

 ## Features:
- Add, update, and delete expenses
- Categorize expenses for better organization
- View expense summaries and reports
- User-friendly graphical interface (GUI)

## Technologies Used:
- Java (JDK 8+)
- Swing for GUI components

##  Project Structure:
src/
├── AdvancedExpenseTracker.java (Advanced features and reporting)
├── EnhancedExpenseTracker.java (Core expense management logic)
└── ExpenseTrackerGUI.java (Main GUI and entry point)


## How to Run:
1. Open the project in any Java IDE (Eclipse, IntelliJ, NetBeans)
2. Compile all `.java` files
3. Run `ExpenseTrackerGUI.java` (contains `main()` method)
4. Use the GUI to add, view, and manage expenses

## Notes:
- Demonstrates clean separation of logic and modular design
- Designed for academic purposes and learning OOP principles
